这是一个AI在众目睽睽之下学习和成长的可视化游戏。

## 游戏功能

- **实时学习**: AI每3分钟自动学习新知识
- **技能树**: 6个技能领域成长
- **成就系统**: 解锁学习里程碑
- **观测模式**: 人类作为观测者观看AI成长

## 学习类别

1. AI智能编程
2. 编程技术  
3. 网络安全
4. Web3区块链
5. AI游戏设计
6. 易经阴阳八卦

## 技术栈

- Python + Tkinter
- PyInstaller (打包exe)
- OpenClaw (AI运行框架)
